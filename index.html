About Me

Hello! My name is Jamie, and I'm a software engineer. I currently work for BBC, more specifically BBC News. I joined BBC as a trainee back in 2015 in their BBC World Service team, and since then I've moved up the ranks to mid-level engineer. I've witnessed and took part in many projects, including migrating from a PHP based stack to using JavaScript with ReactJS, and NodeJS backend.

More recently I've been working on the BBC News website, mainly with the article pages, adding functionality for interactive "includes", as well as moving to a brand new Content Managment System.

If you'd like to know more about my employment history with the BBC or learn more about my programming knowledge, please feel free to use the form below to request my CV!
